
## WeatherApp

Простое погодное приложение на Kotlin для Android.

## Функции
- Текущая погода в Москве
- Почасовой прогноз на текущий день
- Прогноз на 3 дня
- Индикатор загрузки при получении данных
- Обработка ошибок с возможностью повторить запрос

## Технологии
- Kotlin
- Jetpack Compose (UI полностью программный, без XML)
- Retrofit для работы с API
- WeatherAPI.com для получения данных о погоде

## API
Пример запроса:  
`http://api.weatherapi.com/v1/forecast.json?key=YOUR_API_KEY&q=55.7569,37.6151&days=3`

## Запуск
1. Склонировать репозиторий
2. Открыть проект в Android Studio
3. Запустить на эмуляторе или реальном устройстве
